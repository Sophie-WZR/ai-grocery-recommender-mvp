# Image Placeholders

This folder stores screenshots and visual assets used by the project README.

Suggested files:

- `app-home.png` — main Streamlit app screenshot
- `query-example.png` — one example recommendation flow screenshot

Recommended workflow:

1. Run the Streamlit app with:

```bash
streamlit run app.py
```

2. Capture a clean screenshot of the UI.

3. Save it in this folder using one of the filenames above.

4. The top-level `README.md` will automatically reference:

- `docs/images/app-home.png`
- `docs/images/query-example.png`

If the files do not exist yet, GitHub will simply show broken image placeholders until you add them.
